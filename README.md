# Valentine

An interactive Valentine's Day web experience. Features a two-screen flow: a playful ask card where the "No" button evades the cursor, followed by a 14-tile photo calendar that reveals one image per tap — with a lightbox, confetti celebration, and background music.

## Features

- **Ask screen** — YES/NO card with a runaway NO button that dodges the pointer
- **Photo calendar** — 14 tiles, one per day until Valentine's Day; tap each to reveal a photo
- **Lightbox** — full-screen image viewer with keyboard and click-to-close support
- **Confetti** — celebration animation on YES
- **Background music** — autoplays on interaction
- **No backend** — pure HTML/CSS/JS, runs entirely in the browser

## Setup

1. Add 14 photos to `valentine-site/images/` named `01.jpg` through `14.jpg`
2. Open `valentine-site/index.html` in a browser — or deploy to any static host

## Deployment

Deployed via GitHub Pages through `.github/workflows/`.
